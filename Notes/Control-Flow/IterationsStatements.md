**Iteration Statements**
1. For Loops
```
for (int i = 0; i < 10; i++) { }

```

2. Foreach Loops

```
foreach (var item in collection){ }

```
3. While Loops

```
while (true) { }
```
4. Do-While Loops (Run for at least ones)
```
do { Console.WriteLine(); }while (true);

```
Break and Continue
**Break:** Jumps out of the loop
**Continue:** Jumpos to the next iteration
